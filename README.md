# chatbot
Flask‑based chatbot with HTML, CSS, and JavaScript frontend for customer support simulation.
## 🚀 Features
- Clean chat interface with bot and user messages
- Flask backend serving HTML templates and static files
- Responsive design for desktop and mobile
- Easy to extend with custom responses or APIs

## 📂 Project Structure
project/

│── static/

│   ├── style.css

│   └── script.js

│── templates/

│   └── index.html

│── app.py

│── chatbot.py


## ⚙️ Setup Instructions

```bash
# 1. Clone the repository
git clone https://github.com/anshi5/chatbot.git
cd chatbot

# 2. Create and activate a virtual environment
python -m venv .venv
# Linux/Mac
source .venv/bin/activate
# Windows
.venv\Scripts\activate

# 3. Install dependencies
pip install flask

# 4. Run the app
python app.py

# 5. Open in browser
http://127.0.0.1:5000
